# Hidden Bites — UI Direction & Design Prompt

## Product Overview

Hidden Bites is a modern mobile application for discovering and saving underrated cafes, hidden food spots, cozy study places, and local hidden gems nearby.

The app focuses on:

* clean mobile experience
* location-based discovery
* visual storytelling
* modern lifestyle exploration

This is a Flutter MVP project designed for a realistic mobile app experience, not an overly futuristic concept app.

---

# UI Style Direction

The interface should feel:

* clean
* modern
* warm
* minimal
* realistic
* mobile-native

The design should resemble polished modern consumer apps rather than flashy AI-generated concepts.

Avoid:

* excessive gradients
* glassmorphism overload
* neon cyberpunk aesthetics
* complicated layouts
* unrealistic animations
* enterprise dashboard feeling

---

# Visual Inspiration

Inspired by:

* Pinterest
* Instagram
* Lemon8
* Google Maps modern UI
* minimalist lifestyle apps

Primary Accent:

* Pinterest-inspired warm red

Suggested tones:

* soft deep red

* clean white or soft off-white
* optional dark mode later

Text:

* dark gray

Cards:

* rounded corners

---

# Design Principles

## 1. Mobile-first

The UI must feel designed specifically for smartphones:

* thumb-friendly
* bottom navigation focused
* comfortable spacing
* easy scrolling

---

## 2. Content-first

Images and places should become the focus of the interface.

Large place thumbnails are preferred over text-heavy layouts.

---

## 3. Minimal Cognitive Load

The app should feel lightweight and easy to understand immediately.

Avoid:

* too many buttons
* excessive filters
* cluttered navigation

---

# Main Screens

## Home Feed

A clean vertical feed showing hidden places nearby.

Each card contains:

* large image
* place name
* short review
* location preview
* category tag
* save button

Use:

* soft rounded cards
* Pinterest-style content flow
* subtle shadows

---

## Add Hidden Spot

A simple and modern form screen for adding places.

Include:

* image upload
* place name
* short description
* category selector
* auto-detected location
* clean submit button

The layout should feel lightweight and simple.

---

## Explore Map

A modern map exploration screen inspired by Google Maps.

Features:

* nearby place markers
* floating preview cards
* search bar
* clean navigation

Avoid cluttered map overlays.

---

## Place Detail

A clean detail page with:

* hero image
* place information
* short review
* category chips
* save button
* location section

Focus on hierarchy and spacing.

---

## Saved Places

A bookmark collection page with:

* clean cards
* simple organization
* visual browsing experience

---

# Typography

Use:

* modern sans-serif typography
* medium font weights
* comfortable spacing
* minimal text density

The interface should feel calm and readable.

---

# UX Priorities

Prioritize:

* fast interaction
* simple navigation
* visual clarity
* smooth scrolling
* comfortable one-hand usage

---

# Technical Context

This application is a Flutter-based mobile MVP using:

* local-first storage
* geolocation
* image picker
* Google Maps integration

The UI must remain realistic and achievable within a 3-week MVP development timeline.
